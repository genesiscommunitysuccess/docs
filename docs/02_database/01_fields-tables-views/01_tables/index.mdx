Tables are data structures that organise data into rows and columns. 

Generally, the rows consists of records while the columns consist of fields. 

Tables are defined in the file _application-name_**-tables-dictionary.kts**. 

By default, this file located in the folder:<br></br>
**server/**_appname_**/src/main/genesis/cfg/**
