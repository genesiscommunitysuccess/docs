All notable changes to this product will be documented in this page.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.3.2] - 2024-11-07

### Added

- Feature of custom prompt entered by the user.
- New layout-region and layout-item snippets.

### Fixed

- Resolved not adding comments TODO on Genesis View Generation
- Fixed incorrect resourceName attribute for charts.

## [1.3.1] - 2024-10-28

### Added

- Support to FoundationLayout on Genesis View generation.
- Support to the css font-family on Genesis View.

### Changed

- Prompt improvements to generate custom layout and internal components.

### Fixed

- Resolved failing generation sometimes on Genesis View.

## [1.3.0] - 2024-10-23

### Added

- ReqRep on datasources.
- New chart types, including dual axes and stock charts.
- AI button integration for datasources.
- New UI/UX elements and improved snippets.
- Component generation types and creation in Genesis View.
- Dataserver extract and clean-up functionalities.

### Changed

- Improved HTML and CSS generation for more accurate, high-fidelity code output from screenshots, enhancing the precision and quality of the generated front-end structure.
- Enhanced prompt handling for design tokens and grids.
- Added dynamic `cellStyles` for grid components.
- Improved prompts with advanced configuration options for forms.
- Organized components into separate folders and adjusted names.

### Fixed

- Resolved issues with custom-grid-pro styles after framework update.
- Corrected color hex values in styles.
- Cleaned up variable names and fixed typos.
- Fixed playground URL and incremented version number.

### Documentation

- Updated README with new instructions and TODOs.
