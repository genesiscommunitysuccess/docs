DbEntity is the common interface implemented by table entities and view entities.

The DbEntity methods are described below:

|  Name  | Signature | Description |
|---------------|-------------|-------------|
| toDbRecord | `fun toDbRecord(entity: E): DbRecord` | Converts entity to [DbRecord](/reference/developer/api/database/how-to/data-types/dbrecord/) |
| toGenesisSetFormatted | `fun toGenesisSetFormatted(entity: E, configs: Collection<ColumnConfig>? = null): GenesisSet` | converts a view to [GenesisSet](/managing-applications/operate/inter-process-messages/genesis-set/) and applies any formatter/aliases assigned to the fields |
| toGenesisSet | `fun toGenesisSet(entity: E, columns: Collection<String>): GenesisSet` | converts view to [GenesisSet](/managing-applications/operate/inter-process-messages/genesis-set/). This is the plain representation of view fields |
| get | `operator fun get(field: String): Any?` | gets the provided field value |
