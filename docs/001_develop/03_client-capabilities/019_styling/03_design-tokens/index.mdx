This section will cover how to configure tokens relating to different types of styling in your application. Base tokens can be configured which are used in recipes to create different tokens. The compounding effect is a rich system of variables which can be tweaked to alter the look and feel of the UI. This is as well as responding to certain actions, such as building light and dark mode themes.
