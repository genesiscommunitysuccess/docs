# Genesis FDC3

This encapsulates the functionality for working with the FDC3 api. This includes methods for listening for
- Handling fdc3 ready events
- Finding, raising and listening to intents
- Discovering and joining channels
- Broadcasting on and listening to channels

## Overview of `@FDC3`

`@FDC3` is designed to abstract away complexity of working with the fdc3 api.
