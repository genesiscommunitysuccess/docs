# Properties
This component doesn't have any properties which are not also controlled via attributes.

### Slots

| Slot | Use |
|------|-----|
| start | Start slot. |
| end | End slot. |

### Parts
| Part    | Description                                       |
|---------| ------------------------------------------------- |
| control | The button element. |
| content | The element wrapping button content. |

### Events fired

| Event | Use |
|---|---|
| click | Emits a change event when the button is pressed |

### Events listened to

This component doesn't listen to any events.

### Further details
This component is an implementation of an HTML [button](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/button) element.
