# Foundation Header

`foundation-header` is a micro front-end and a semi-batteries-included component.
It consists of a navigation bar and a flyout menu, with routing, account-logout capabilities, and the ability to change the current language.

## Examples

### Adding a header component to the template

Here is an example of a simple navigation bar with two navigation items. This header does not show additional control buttons but displays the default ones, allowing for showing submenus, the logged-in user's name, and the connection status.

<Tabs defaultValue="genesis" values={[{ label: 'Genesis', value: 'genesis', }, { label: 'React', value: 'react', }, { label: 'Angular', value: 'angular', }]}>

<TabItem value="genesis">

```html
  <foundation-header></foundation-header>
```

Usage:
```typescript
@customElement({
  name: 'header-example',
  template: html`
    <foundation-header
      :routeNavItems=${(x) => x.navItems}
    ></foundation-header>
  `,
})
export class HeaderExample extends GenesisElement {
  const navItems = [{
    title: "Home",
    icon: {
      name: "home",
      variant: "solid",
    },
    navId: "header",
    placementIndex: 0,
    routePath: "home",
    routeName: "home",
  },
  {
    title: "Grids",
    icon: {
        name: "table",
        variant: "solid",
      },
    navId: "header",
    placementIndex: 1,
    routePath: "grids",
    routeName: "grids",
  }];
}
```
