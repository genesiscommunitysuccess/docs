This component enables you to show an icon.

Use cases:
* displaying a specified icon
* highlighting items in a listbox
* cell-renderers and many others

:::note
You can use any free publicly-available icon from fontawesome. These are available via the links below:
- [solid/regular](https://fontawesome.com/search?o=r&m=free) icons
- [brand](https://fontawesome.com/search?o=r&m=free&f=brands) icons

Paying for fontawesome pro enables you to use all the fontawesome icons with this component.
:::

    schemas: [CUSTOM_ELEMENTS_SCHEMA],

## API

This table shows attribute binding examples for Genesis Component syntax. The HTML closing tag is not included.

### Attributes

<table>
  <thead><tr><th>Name</th><th>Type</th><th>Description</th><th>Example</th></tr></thead>
  <tbody>
    <tr>
      <td>variant</td>
      <td><code>"solid" | "regular" | "brand"</code></td>
      <td>`solid` is the default, `regular` usually is slightly more finely drawn, `brand` is for brand icons only.</td>
<td>

```typescript
<rapid-icon variant="regular">
```

</td>
    </tr>
    <tr>
      <td>size</td>
      <td><code>"2xs" | "xs" | "sm" | "lg" | "xl" | "2xl" | "1x" | "2x" | "3x" | "4x" | "5x" | "6x" | "7x" | "8x" | "9x" | "10x"</code></td>
      <td>Controls the size of the icon. Default: "sm"</td>
<td>

```typescript
<rapid-icon size="xl">
```

</td>
    </tr>
    <tr>
      <td>name</td>
      <td><code>string</code></td>
      <td>The name of the icon (matching with the name in fontawesome).</td>
<td>

```typescript
<rapid-icon name="iconName">
```

</td>
    </tr>
    <tr>
      <td>error</td>
      <td><code>boolean</code></td>
      <td>Adds error styling to the icon when `tooltip` attribute is also set.</td>
<td>

```typescript
<rapid-icon error>
```

</td>
    </tr>
    <tr>
      <td>tooltip</td>
      <td><code>string</code></td>
      <td>The tooltip for the icon. If not set, there is no tooltip.</td>
<td>

```typescript
<rapid-icon tooltip="Icon tooltips">
```

</td>
    </tr>
    <tr>
      <td>tooltip-id</td>
      <td><code>string</code></td>
      <td>The tooltip id.</td>
<td>

```typescript
<rapid-icon tooltip-id="Tooltip_ID">
```

</td>
    </tr>
  </tbody>
</table>
