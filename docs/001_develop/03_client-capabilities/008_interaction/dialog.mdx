Anchors add different types of links to your page.

Use cases:
* Creating a hyperlink to a different page
* Linking to a specific region of the current page
* An email link which pre-populates the recipient field

      template: `
        <rapid-button (click)="openDialog()">
          Open Dialog
        </rapid-button>

        <rapid-dialog
          #dialogRef
          id="example1"
          class="example-dialog"
          aria-label="Simple modal dialog"
          modal="true"
          hidden
        >
          <h2>Dialog</h2>
          <p>This is an example dialog.</p>
        </rapid-dialog>
      `,
      standalone: true,
      schemas: [CUSTOM_ELEMENTS_SCHEMA],
      imports: [FormsModule],
    })
    export class MyComponent {
      @ViewChild('dialogRef') dialogRef!: ElementRef;

      openDialog() {
        this.dialogRef.nativeElement?.removeAttribute('hidden');
        this.dialogRef.nativeElement?.show();
      }
    }

### Properties

This component doesn't have any properties which are not also controlled via attributes.

### Methods

|Method|Description|Example|
|--|--|--|
|show|Show the dialog|`dialogRef.show()`|
|hide|Hides the dialog|`dialogRef.hide()`|

### Slots

|Slot|Description|
|---|---|
|Default|All the content which you want to display on the dialog box when shown.|

### Parts

| Part | Description |
| --- | --- |
| positioning-region | A wrapping element used to center the dialog and position the modal overlay |
| overlay | The modal dialog overlay |
| control | The dialog element |
