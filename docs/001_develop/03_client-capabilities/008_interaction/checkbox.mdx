# Checkbox

A form-connected implementation of a [checkbox](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/checkbox) input.

Use cases:
* Multi-selection forms
* Filtering
* Sorting
* To-Do lists
* Privacy and consent

Any use where the user is to give a boolean type input choice.

@customElement({
  name: 'my-element',
  template: html`
    <rapid-checkbox
      ?disabled="${(x) => x.disabled}"

  `,

  @observable isSelected = false;
  checkChanged() {

import { Component, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';

@Component({

  template: `
    <rapid-checkbox
      name="isSelected"
      [checked]="isSelected"
      (change)="onCheckboxChange($event)"
    ></rapid-checkbox>
  `,
  styleUrls: ['./my.component.css'],
  standalone: true,
  schemas: [CUSTOM_ELEMENTS_SCHEMA],
  imports: [FormsModule],
})
export class AppComponent {
  isSelected: boolean = false;
  onCheckboxChange(event: Event) {
    const target = event.target as HTMLInputElement;
    this.isSelected = target.checked;
    console.log(this.isSelected);
  }
}

## DOM API

Property and attribute binding examples for Genesis Component syntax.

| Attribute/Property | Type | Description |
|---|---|---|
|checked|`boolean`|The boolean value of whether the checkbox is checked. |
|readonly|`boolean`|Configures the component so that the user cannot alter the state via the web browser.|
|disabled|`boolean`|Makes the component visually opaque, and configures it so that the user cannot alter the state via the web browser.|
|form|`string`|Associates this component with a form. The Form id must be passed. If no Id is passed, then it will be associated with the ancestor form.|

### Examples (closing tag omitted):

Attributes:
```typescript
<rapid-checkbox checked>
<rapid-checkbox readonly>
<rapid-checkbox disabled>
<rapid-checkbox form="myForm">
```
Properties:
```typescript
<rapid-checkbox ?checked="${sync((x) => x.isSelected, 'boolean')}">
<rapid-checkbox ?readonly="${(x) => x.readOnly}">
<rapid-checkbox ?disabled="${(x) => x.disabled}">
<rapid-checkbox :form="${(x) => x.myForm}">
```

### Slots

| Slot | Use |
|---|---|
| Default | The default slot for the label. |
| checked-indicator | The checked-state indicator. |
| indeterminate-indicator | The indeterminate (non-checked) indicator. |

### Parts

| Part | Use |
|---|---|
| control | The element representing the visual checkbox control.|
| label | The label text.|

### Events fired

| Event | Use | Example |
|---|---|---|
| change | Emits a custom change event when the checked state changes. Access the value on the event via `.target.checked`.|`<rapid-checkbox @change="${(x) => x.myChangeHandler(event)}">`|

### Events listened to

This component doesn't listen to any events.
