# Stepper

Use cases:
* Provide a multi-step process or dynamic form that the user can follow.

Rapid Stepper requires `rapid-stepper-tab` and `rapid-stepper-tab-panel` in order to work properly.

## API

Property and attribute binding examples for Genesis Component syntax.

| Name     | Type      | Description                                                                      | Examples |
|----------|-----------|----------------------------------------------------------------------------------|----------|
| hide-submit-button | `boolean` | Hides or shows the submit button | `<rapid-stepper hide-submit-button>` |
| orientation | `string` | There are two stepper orientations to choose from: vertical and horizontal. | `<rapid-stepper orientation="horizontal">` |

### Slots

| Slot |
|---|
| tab	|
| tab-panel |
| action-buttons |

### Parts

| Part |
|---|
| stepper-tab |
| stepper-tab-menu |
| stepper-tab-progress |
| progress-ring |
| stepper-tab-panel |
| stepper-panel-container |
| action-buttons-container |

### Events fired

| Name    | Description                                 |
|---------|---------------------------------------------|
| @submit | Possibility to pass the function to be performed in the last step after clicking submit. |

### Events listened to
This component doesn't listen to any events.
