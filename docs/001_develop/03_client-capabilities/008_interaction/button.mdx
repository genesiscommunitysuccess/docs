# Button

Implementation of a [button](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/button) component.

Use cases:
* Submit forms
* Trigger actions
* Create interactive elements
* Set options

## API

Property and attribute binding examples for Genesis Component syntax.

| Attribute/Property | Type | Description |
|---|---|---|
|autofocus|`boolean`|The boolean value of whether the checkbox is checked. |
|appearance|`string`|Configures the component so the user cannot alter the state via the web browser.|
|disabled|`boolean`|Same functionality as `readonly`, and additionally adds visual opacity to the component.|
|form|`string`|Associates this component to a form. Form id needs to be passed. If no Id informed, then it will be associated with the ancestor form.|
|name|`string`|Defines the name of the element.|
|type|`string`|Defines the mode of the button.|
|value|`string`|Defines the value associated with name.|

The type attribute can be:
* button (default) - performs an action when clicked
* reset - can influence a group of components associated with the same form.
