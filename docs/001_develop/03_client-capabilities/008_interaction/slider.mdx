Use cases:
* Interact with data visualization
* Input for numeric values
* Dynamic filters

## API

Property and attribute binding examples for Genesis Component syntax.

| Name        | Type      | Description                                                   | Examples |
|-------------|-----------|---------------------------------------------------------------|---|
| disabled    | `boolean` | Similar to `readonly`, but with a blur layer in the component. | `<rapid-slider disabled>` |
| form        | `string`  | Associates this component to a form. Form `id` needs to be passed. If no Id informed, then it will be associated with the ancestor form. | |
| max         | `number`  | Defines maximum number of the slider.                          | `<rapid-slider max=50>` |
| min         | `number`  | Defines minimum number of the slider.                          | `<rapid-slider min=10>` |
| step        | `number`  | Defines the value of each step of the slider.                  | `<rapid-slider step=5>` |
| readonly    | `boolean` | The slider is for display only, and cannot be changed by the user. | `<rapid-slider readonly>` |
| value       | `number`  | Defines a value for the component when it is created.          | `<rapid-slider value=20>` |
| orientation | `string`  | Defines the orientation: `vertical` or `horizontal`.           | `<rapid-slider orientation="horizontal">` |

### Slots

| Slot | Use |
|---|---|
| checked-indicator	| The checked indicator. |
| track |	The track of the slider. |
| track-start	| The track-start visual indicator. |
| thumb	| The slider thumb. |

### Parts

| Part | Description |
|---|---|
| control |	The element representing the visual radio control. |
| label |	The label. |
| positioning-region | The region used to position the elements of the slider. |
| track | The region containing the track elements. |
| track-start | The element wrapping the track start slot. |
| thumb-container | The thumb container element which is programmatically positioned. |
| thumb	| The thumb element. |

### Events fired
| Name    | Description                                 |
|---------|---------------------------------------------|
| @change | Fires the event when the slider component changes its state. |

### Events listened to
This component doesn't listen to any events.

### Further details
This component is an implementation of an HTML [range slider](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/Input/range) element.
