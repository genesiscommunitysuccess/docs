Anchors enable you to links of different types of in a page.

Use cases:
* a hyperlink to a different page
* linking to a specific region of the current page
* an email link that pre-populates the recipient field

## Example

## API

### Attributes

<table>
    <tr>
        <th>Attribute</th>
        <th>Type</th>
        <th>Description</th>
        <th>Example</th>
    </tr>
    <tr>
        <td>download</td>
        <td><code>string | boolean</code></td>
        <td>Causes the browser to treat the linked URL as a download. This attributes only works for same-origin URLs, or when using the <code>blob:</code> and <code>data:</code> schemes. Different browsers behave slightly differently for downloads, such as asking the user where to save the file or automatically saving to a downloads directory.</td>
<td>

```typescript
<rapid-anchor href="files/genesis.pdf" download>
```

</td>
    </tr>
    <tr>
        <td>href</td>
        <td><code>string</code></td>
        <td>The URL that the hyperlink points to. This is commonly to another page in your app, but it could be to a region of the current page. You can also use schemes such as <code>mailto:</code>, <code>javascript:</code>, <code>tel:</code>, and <code>sms:</code></td>
<td>

```typescript
<rapid-anchor href="https://genesis.global/">
```

</td>
    </tr>
    <tr>
        <td>hreflang</td>
        <td><code>string</code></td>
        <td>A hint to the language of the linked document. Doesn't add any functionality itself.</td>
<td>

```typescript
<rapid-anchor href="https://genesis.global/" hreflang="en">
```

</td>
    </tr>
    <tr>
        <td>ping</td>
        <td><code>string</code></td>
        <td>A space-separated list of URLs to which to send a <code>PING</code> message. This is commonly used for tracking.</td>
<td>

```typescript
<rapid-anchor ping="https://genesis.global/">
```

</td>
    </tr>
    <tr>
        <td>referrer</td>
        <td><code>string</code></td>
        <td>How much of <a href="https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Referer">the referrer</a> to send when the user clicks the link.</td>
<td>

```typescript
<rapid-anchor href="https://genesis.global/" referrer="no-referrer">
```

</td>
    </tr>
    <tr>
        <td>rel</td>
        <td><code>string</code></td>
        <td>The relationship of the linked URL to the current page. Examples are the next or previous page, the author's page, or to help documentation.</td>
<td>

```typescript
<rapid-anchor href="https://genesis.global/author" rel="author">
  Author's page
</rapid-anchor>
```

</td>
    </tr>
    <tr>
        <td>target</td>
        <td><code>"_blank" | "_self" | "_parent"</code></td>
        <td>Where to display the linked URL. Most common use is to set <code>_blank</code> for making the link open in a new tab or window.</td>
<td>

```typescript
<rapid-anchor href="https://genesis.global/" target="_blank">
```

</td>
    </tr>
    <tr>
        <td>type</td>
        <td><code>string</code></td>
        <td>A hint to the <a href="https://developer.mozilla.org/en-US/docs/Glossary/MIME_type">media type</a> of the linked document. Doesn't add any functionality itself.</td>
<td>

```typescript
<rapid-anchor href="https://genesis.global/" type="application/pdf">
```

</td>
    </tr>
</table>

### Properties

The properties of this component are all controlled via [attributes](#attributes).

### Slots

|Slot|Description|
|---|---|
|Default| The elements that are contained in the anchor region, and which trigger the `href` when clicked.|
|start| Optional content to be placed before the anchor content.|
|end| Optional content to be placed after the anchor content.|

### Parts

| Part | Description |
| --- | --- |
| control | The anchor element |
| content | The element wrapping anchor content

### Events fired

This component doesn't emit any events.

### Events listened to

This component doesn't listen to any events.
