# Criteria Segmented Control

This version allows you to use a segmented control that generates criteria based on the selected item.

Use cases:
* Visually and structurally group related interactive criteria.

          { label: 'C', field: 'INSTRUMENT_ID', value: 'id3', serialiser: Serialisers.EQ },

        handleCriteriaChange(event: Event) {

## API

Properties:

| Name             | Type    | Description                                          | Example |
|-----------------|----------|------------------------------------------------------|---------|
| criteriaOptions | array    | Array of criteria objects that can be selected       |         |
| value           | `string` | The value of the constructed criteria expression     |         |
| selectedLabel   | `string` | The selected options label                           |         |

### Slots

| Slot | Description |
|---|---|
| label | The label |

### Parts

There are no parts.

### Events fired

| Part | Type | Use | Example |
|---|---|---|---|
| change | | Emits a custom change event when the selection state changes.| |


### Events listened to

This component doesn't listen to any events.
