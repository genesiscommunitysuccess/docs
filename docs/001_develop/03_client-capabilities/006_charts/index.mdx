# Genesis G2Plot Chart

Genesis charts are based on [G2plot](https://g2plot.antv.antgroup.com/en/), a powerful charting library that provides various customizable chart types, enabling users to create interactive and visually appealing data visualizations.

The `G2PlotChart` component comes with several key properties that allow you to configure and display the chart with your desired data and settings. These properties give you full control over how the chart behaves and looks:

- [`config`](./docs/api/g2plot-chart.g2plotchart.config.md) - The configuration to use for the chart. Check [ChartConfig](./docs/api/g2plot-chart.chartconfig.md) for possible options. This allows for detailed customization of the chart's appearance, behavior, and functionality.

- [`data`](./docs/api/g2plot-chart.g2plotchart.data.md) - The data to display in the chart. This is the dataset that will be visualized, and it can be dynamically updated based on user interaction or data source changes.

- [`type`](./docs/api/g2plot-chart.g2plotchart.type.md) - The type to use for the chart. If not set, the default bar chart type will be used. You can specify different chart types such as line, area, pie, and others to best match your data and analytical needs.

These properties, combined with the wide range of chart types available, offer a flexible way to present data visually and interactively, making complex information more accessible and easier to analyze.

@Component({
  selector: 'my-root',
