# Genesis Foundation UI Criteria Utils


## Installation

To enable this module in your application, follow the steps below.

1. Add `@genesislcap/foundation-criteria` as a dependency in your `package.json` file. Whenever you change the dependencies of your project, ensure you run the `$ npm run bootstrap` command again. You can find more information in the [package.json basics](../../../../../web/basics/package-json-basics/) page.

```json
{
  ...
  "dependencies": {
    ...
    "@genesislcap/foundation-criteria": "latest"
    ...
  },
  ...
}
```
