# Grid Tabulator

[![lerna](https://img.shields.io/badge/maintained%20with-lerna-cc00ff.svg)](https://lerna.js.org/)
[![TypeScript](https://img.shields.io/badge/%3C%2F%3E-TypeScript-%230074c1.svg)](https://www.typescriptlang.org/)

This package contains the `grid-tabulator` component which wraps tabulator grid. It also includes a `grid-tabulator-client-side-datasource` component that fetches data from a Genesis server (or any other server that implements the Genesis protocol).

## Installation

To enable this module in your application, follow the steps below.

1. Add `@genesislcap/grid-tabulator` as a dependency in your `package.json` file. Whenever you change the dependencies of your project, ensure you run the `$ npm run bootstrap` command again. You can find more information in the [package.json basics](../../../../web/basics/package-json-basics/) page.

```json
{
  "dependencies": {
    "@genesislcap/grid-tabulator": "latest"
  },
}
```
