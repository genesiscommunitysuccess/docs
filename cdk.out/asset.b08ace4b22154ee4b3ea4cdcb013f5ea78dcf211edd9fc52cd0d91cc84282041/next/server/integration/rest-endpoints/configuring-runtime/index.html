REST endpoints - Configuring Runtime

Introduction | Where to define | Basics | Advanced | Configuring runtime | Testing

There are two important files in your application that contain configuration information:

  • application-name-processes.xml
  • application-name-service-definitions.xml

REST endpoints require that both their resources, and the Genesis Router are configured within the above files.
