REST endpoints - Where to Define

Introduction | Where to define | Basics | Advanced | Configuring runtime | Testing

The Genesis platform makes defining REST Endpoints easy. Instead of needing to configure individual endpoints, you only have to declare the relevant resources, and configure the Genesis Router.

The following resources support automatic route registration:

  • Data Servers
  • Request Servers
  • Event Handlers
