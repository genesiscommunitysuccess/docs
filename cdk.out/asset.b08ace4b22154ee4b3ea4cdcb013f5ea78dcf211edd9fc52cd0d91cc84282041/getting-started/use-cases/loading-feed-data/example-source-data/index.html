Loading feed data - Example source data

  • There are clear markers for START-OF-FILE and END-OF-FILE.
  • There are clear markers for START-OF-DATA and END-OF-DATA. The data itself does not start until a long way into the file.
  • Once you reach the data, each issuance is on a separate row, and the fields are separated by pipes.
